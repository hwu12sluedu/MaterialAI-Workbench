# 复合材料机器学习：任务定义与可信验证

这份文档回答三个最容易在项目介绍和面试中说混的问题：

1. 机器学习究竟预测什么？
2. 为什么有时用朴素贝叶斯，有时用随机森林或神经网络？
3. 怎样证明模型准确，而不是只展示一张拟合图？

## 1. 先记住结论

复合材料机器学习至少包含两类不同问题。

| 问题 | 输出例子 | 推荐起点 | 主要指标 |
|---|---|---|---|
| 分类 | 基体开裂、分层、纤维断裂；合格/不合格 | 朴素贝叶斯、逻辑回归、随机森林 | Macro-F1、Balanced Accuracy、混淆矩阵 |
| 回归 | `E2`、`G23`、强度、孔边应力、CAI 强度 | 线性/物理基线、随机森林、SVR、GPR、MLP | MAE、RMSE、R2、相对误差 |

**朴素贝叶斯不能直接回答“弹性模量是多少 MPa”。** 它最适合回答“这条信号或这个工况属于哪一种损伤类别”。若把连续强度强行切成高、中、低三档，只是为了使用分类算法，会丢失信息，也会引入人为阈值。

本项目第一条生产主线仍是回归：

```text
三相微观结构与组分参数
-> Abaqus RVE/PBC
-> 等效弹性参数
-> 三维带孔板
-> ODB 工程响应
-> 回归代理模型
-> 未见工况复核
```

分类是第二条数据主线：

```text
试验传感特征或可信损伤变量
-> 人工/试验标注的损伤模式
-> 朴素贝叶斯基线
-> RF/SVM/其他模型对比
-> 独立试样验证
```

## 2. 项目最终能够预测什么

### 2.1 第一阶段：单向纤维复材均匀化性能

输入：

- 纤维体积分数、半径、方向和空间分布；
- 纤维、界面、基体材料参数；
- RVE 尺寸、网格和随机种子。

输出：

- `E1/E2/E3`；
- `G12/G13/G23`；
- `nu12/nu13/nu23`。

标签优先来自 Abaqus 六工况 PBC 的 ODB，而不是混合定则估算。模型首先比较物理基线、Random Forest 和 MLP；样本量与噪声允许时再增加 Gaussian Process、梯度提升和图像模型。

### 2.2 第二阶段：三维带孔板结构响应

输入：

- 上一步的正交各向异性性能；
- 板长、板宽、厚度、孔径、铺层或材料方向；
- 载荷、边界条件和网格参数。

输出：

- 孔边最大应力；
- 最大位移和反力；
- Hashin/Tsai-Wu 等失效指标；
- 后续的损伤起始载荷和极限载荷。

代理模型只用于快速筛选。超出训练域、预测不确定性高或接近失效边界的方案，必须回到 Abaqus 求解。

### 2.3 第三阶段：复材损伤分类

朴素贝叶斯的首个合理任务是 `damage_mode`：

```text
matrix_cracking       基体开裂
delamination          分层
fiber_breakage        纤维断裂
interface_debonding   界面脱粘，可在数据允许时加入
```

可用输入分两种来源：

- 实验：声发射幅值、频率、持续时间、能量、载荷水平等；
- 仿真：Hashin 各模式变量、界面损伤变量、局部应力应变与载荷历程。

仿真损伤变量只能证明“代理模型学会了复现该有限元模型”，不能自动证明真实材料损伤识别准确。最终仍需独立试样、显微观察、DIC、CT 或声发射标注验证。

## 3. 为什么选择这些算法

| 算法 | 在本项目中的角色 | 优点 | 主要限制 |
|---|---|---|---|
| Gaussian Naive Bayes | 损伤模式分类基线 | 快、概率输出、容易解释 | 假定条件独立，复材传感特征往往相关 |
| Logistic Regression | 分类线性基线 | 系数可解释、便于排查泄漏 | 难表达复杂非线性边界 |
| Random Forest | 表格回归与分类主基线 | 小中型数据稳健、可看特征重要性 | 外推弱，树间方差不等于完整可信区间 |
| Gradient Boosting | 高质量表格数据对比 | 非线性拟合强 | 调参与过拟合控制更重要 |
| SVR/SVM | 小样本回归或分类对比 | 高维小样本常有效 | 缩放和超参数敏感 |
| Gaussian Process | 小样本回归与不确定性 | 可给出预测分布 | 样本增大后计算成本高 |
| MLP | 非线性多输出代理模型 | 可扩展到多任务和深度学习 | 小样本不一定优于树模型 |
| CNN/GNN | 微观图像、体素或网格场 | 能学习空间拓扑 | 需要更多真实标签和更严格的域外验证 |

面试中不要说“神经网络一定比传统模型准”。正确说法是：**以物理基线和简单机器学习为下限，通过受控划分比较模型，再依据误差、稳定性、速度和不确定性选择。**

## 4. 外部证据矩阵

项目内置登记表位于：

```text
material_ai_workbench/resources/composite_benchmarks.json
```

它把论文报告值与本站复现值分开。当前登记项均未标记为 `reproduced`。

### 4.1 有限元或数值均匀化标签

1. Pathan 等使用 1800 个单向纤维微观结构、有限元标签、二点相关函数、PCA 和梯度提升树，预测宏观刚度与横向屈服强度；论文报告误差量级不超过 5%。这与本项目“微观结构 -> RVE/PBC -> 代理模型”最接近。DOI：[10.1038/s41598-019-50144-w](https://www.nature.com/articles/s41598-019-50144-w)。
2. 王吉玲等把周期 RVE 与 ANN 结合，覆盖空间随机、层内随机和定向短纤维，输入包括长度、长径比、体积分数和组分属性；摘要报告三个代理模型的 `R2` 均高于 0.98。DOI：[10.13801/j.cnki.fhclxb.20240229.005](https://fhclxb.buaa.edu.cn/article/doi/10.13801/j.cnki.fhclxb.20240229.005)。
3. Pasha 等使用周期微观结构、材料属性编码和物理上下界预测 `E22/E33/G23`，数据范围覆盖 `Vf=25%-75%` 和较宽的组分模量比。它支持本项目前进到体素/CNN 时加入物理可接受性检查。DOI：[10.1038/s41598-023-34823-3](https://www.nature.com/articles/s41598-023-34823-3)。

这些论文证明“有限元可生成代理模型标签”的研究路线成立，但不能证明我们的 Abaqus 建模、边界条件和材料参数已经准确。

### 4.2 实验性能回归

1. Shah 等使用 NCAMP 单向碳纤维层合材料实验数据，把环境和固化循环纳入输入，论文报告模量预测最高 `R2=0.98`。DOI：[10.1039/D2MA00698G](https://pubs.rsc.org/en/content/articlelanding/2022/ma/d2ma00698g)。
2. Alsheghri 等制造并测试了 62 个、九类 CFRP 试样，比较 Ridge、Random Forest 和 SVR；论文报告弯曲强度、弯曲模量和 II 型能量释放率的 `R2` 分别为 0.966、0.871 和 0.903。文章与数据均公开：[论文](https://pmc.ncbi.nlm.nih.gov/articles/PMC11975104/)、[Mendeley 数据](https://data.mendeley.com/datasets/fspdwb4mst/1)。
3. Liu 等公开了碳-环氧斜纹编织复材的拉伸、剪切和压缩曲线、强度、模量和泊松比，可用于 Abaqus 材料卡校准与外部验证。DOI：[10.1016/j.dib.2022.108650](https://doi.org/10.1016/j.dib.2022.108650)。
4. Hasebe 等公开了不同铺层、冲头形状和冲击能量下的 CFRP 冲击后压缩数据，适合后续预测 CAI 强度与强度降低率。DOI：[10.1016/j.dib.2025.111509](https://pubmed.ncbi.nlm.nih.gov/40235700/)。

### 4.3 朴素贝叶斯与损伤分类

1. Addin 等在 AS4/3501-6 准各向同性石墨-环氧层合板上，从 Lamb 波幅值特征分类人工损伤，说明朴素贝叶斯可用于复材离散损伤检测。DOI：[10.1016/j.matdes.2006.07.018](https://doi.org/10.1016/j.matdes.2006.07.018)。
2. Amoateng-Mensah 等比较多种分类模型，用声发射和载荷特征识别纤维断裂、分层和基体开裂。论文中的朴素贝叶斯对分层的 F1 为 0.96，但对纤维断裂和基体开裂分别只有 0.63 和 0.74，正好说明“能用”不等于“最优”。DOI：[10.3390/ma19102091](https://www.mdpi.com/1996-1944/19/10/2091)。

## 5. 三栏结果制度

任何模型页面和报告最终都应显示三类值：

| 栏位 | 含义 | 能否作为本站结论 |
|---|---|---|
| Literature reported | 原论文在其数据和划分上的结果 | 不能，只是外部参照 |
| Workbench reproduced | 本仓库固定数据快照和代码得到的结果 | 可以，但仅适用于该数据域 |
| Abaqus/experiment holdout | 未参与训练的 Abaqus 工况或实验试样结果 | 最关键的工程验证 |

只有同时保存以下内容，登记状态才能从 `data_available` 改为 `reproduced`：

- 数据来源、许可、下载日期和文件哈希；
- 特征和标签定义、单位、缺失值处理；
- 固定训练/验证/测试分组；
- 模型参数、随机种子和代码提交号；
- 每条测试样本的真值与预测值；
- 指标、残差图、失败案例和适用范围；
- 一条可重复执行的命令或测试。

## 6. 正确的数据划分

复材数据很容易发生“看起来很准”的泄漏。

错误例子：同一 RVE 的六个载荷工况、同一块板切出的多个试样、同一铺层的重复测量被随机分到训练和测试两侧。

正确原则：

- RVE 数据按 `microstructure_id` 或随机种子分组；
- 宏观模型按几何族或 DOE 设计点分组；
- 试验数据按母板、批次或材料类型分组；
- 时间序列按完整 Job 分组，不能逐帧随机切分；
- 最后保留至少一个材料体系、铺层族或参数区间做域外测试。

若论文使用普通随机划分，而本站使用更严格的分组划分，本站指标可能更低。这通常更可信，不是项目失败。

## 7. 指标应该怎样看

### 7.1 回归

- `MAE`：平均差多少工程单位，最直观；
- `RMSE`：对少数大误差更敏感；
- `R2`：解释方差，不能替代绝对误差；
- 相对误差分位数：至少报告 P50、P90 和最大值；
- 物理违规率：负模量、超出上下界等不合理预测比例；
- 区间覆盖率：声称 90% 预测区间时，真值落入区间的比例。

### 7.2 分类

- `Macro-F1`：各损伤模式同等重要；
- `Balanced Accuracy`：类别不均衡时优于普通准确率；
- 混淆矩阵：看清基体开裂被错成纤维断裂的频率；
- 每类 Precision/Recall：工程上漏报和误报代价不同；
- 概率校准：`0.9` 置信度是否真的约有 90% 正确。

不要只报 Accuracy，也不要只挑最好的一次随机种子。

## 8. 本项目的五级验证链

### Level 1：代码与数据正确性

- 单位、范围、NaN、重复样本、单调性和刚度正定检查；
- 每行数据可追溯到 Case、Job、INP、ODB 和提取脚本。

### Level 2：物理基线

- 弹性性能与 Voigt/Reuss、混合定则或解析解对照；
- 带孔板在适用条件下与经典应力集中结果对照；
- 代理模型必须优于均值预测和简单物理基线。

### Level 3：Abaqus 留出验证

- 训练时完全不见的微观结构和几何参数；
- 重新提交 Abaqus，而不是复用训练 CSV；
- 比较 ODB 真值、误差、不确定性和计算加速比。

### Level 4：公开实验或论文数据复现

- 首个候选是 62 个 CFRP 试样的公开数据；
- 第二个候选是编织复材完整面内试验数据；
- 记录本站指标，不抄论文数字。

### Level 5：自己的实验或真实项目案例

- 用你日常积累的 Abaqus 案例和可公开/脱敏实验数据；
- 留出完整材料批次或产品族；
- 最终形成“仿真、实验、机器学习”三方误差预算。

## 9. 第一个真正可复现的实验题目

题目：**用公开 CFRP 实验数据预测弯曲强度、弯曲模量和 II 型能量释放率。**

执行顺序：

1. 下载 Mendeley 数据并保存原始文件哈希；
2. 建立四个输入变量和三个回归目标的数据字典；
3. 按 CFRP 类型做 GroupKFold 或留一类型验证；
4. 先跑均值、Ridge，再跑 RF、SVR 和 MLP；
5. 保存每一折预测，不只保存平均 `R2`；
6. 分析玻璃化转变温度、夹层和制造压力的影响；
7. 与论文报告值并列展示，解释划分差异；
8. 把表现最差的 CFRP 类型作为失败案例分析。

朴素贝叶斯不加入这个回归实验。它将在有可信 `damage_mode` 标签的声发射/损伤数据集上单独比较。

## 10. 如何读取内置基准

```python
from material_ai_workbench.composite_benchmarks import composite_benchmark_rows

regression_data = composite_benchmark_rows(
    task_kind="regression",
    reproduction_status="data_available",
)

for row in regression_data:
    print(row["material_system"], row["targets"], row["dataset_url"])
```

状态定义：

- `reference_only`：只登记论文；
- `data_available`：确认数据入口，尚未复现；
- `reproduced`：本站证据包完整并通过测试。

## 11. 你必须能讲清的五句话

1. 我的代理模型不是替代所有有限元，而是替代已验证参数域内的大量重复求解。
2. 回归模型预测模量、强度和结构响应；朴素贝叶斯是损伤模式分类基线。
3. 训练标签分为解析估算、Abaqus PBC/ODB 和实验三种保真度，不能混着宣称精度。
4. 我按微观结构、材料批次或几何族分组验证，避免同源样本泄漏。
5. 产品显示文献值、本站复现值和独立 Abaqus/实验验证值，并对域外输入提示复核。
